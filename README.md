# 辩论赛流程控制系统

一个基于纯前端的辩论赛流程控制系统，用于管理40分钟标准辩论赛的时间和流程。

## 功能特点

### 📋 完整的辩论赛流程
- **第一阶段：开篇立论** (6分钟)
  - 正方一辩立论 (3分钟)
  - 反方一辩立论 (3分钟)

- **第二阶段：攻辩/申论** (8分钟)
  - 正方二辩申论/攻辩 (2分钟)
  - 反方二辩申论/攻辩 (2分钟)
  - 正方三辩申论/攻辩 (2分钟)
  - 反方三辩申论/攻辩 (2分钟)

- **第三阶段：自由辩论** (8分钟)
  - 双方各4分钟，可自由切换发言方
  - 实时显示正反双方剩余时间

- **第四阶段：总结陈词** (6分钟)
  - 反方四辩总结陈词 (3分钟)
  - 正方四辩总结陈词 (3分钟)

- **第五阶段：互动环节** (4分钟)
  - 观众观点发言 (2名，共4分钟)

- **第六阶段：评委点评** (6分钟)
  - 评委点评 (5分钟)
  - 结果公布 (1分钟)

### ⏱️ 计时功能
- **精确计时**：秒级精确的倒计时
- **双计时器**：自由辩论阶段支持正反双方独立计时
- **进度条显示**：直观显示当前环节剩余时间
- **时间警告**：剩余时间不足时自动高亮提醒
- **总用时统计**：实时显示整场辩论赛已用时间

### 🎮 流程控制
- **开始/暂停/重置**：灵活控制计时器
- **阶段跳转**：可快速跳转到任意阶段
- **下一环节**：自动进入下一个子环节或阶段
- **发言方切换**：自由辩论时一键切换正反方

### 🎨 界面设计
- **现代化UI**：渐变背景，卡片式布局
- **响应式设计**：支持桌面和移动设备
- **可视化反馈**：
  - 阶段进度指示器
  - 当前环节高亮显示
  - 已完成环节标记
  - 时间警告动画效果

## 使用方法

### 安装
无需安装任何依赖，直接在浏览器中打开即可使用。

### 运行
1. 克隆或下载本项目
2. 用浏览器打开 `index.html` 文件
3. 开始使用

### 操作说明

#### 基本操作
1. **开始计时**：点击"开始"按钮启动当前环节计时
2. **暂停计时**：点击"暂停"按钮暂停计时
3. **重置计时**：点击"重置"按钮将当前环节时间重置
4. **下一环节**：点击"下一环节"进入下一个子环节

#### 阶段跳转
点击顶部的阶段指示器可快速跳转到指定阶段（如正在计时会提示确认）

#### 自由辩论特殊操作
在自由辩论阶段：
- 系统显示双计时器，分别显示正反方剩余时间
- 点击"切换发言方"按钮切换当前发言方
- 当前发言方的计时器会高亮显示
- 任意一方时间用完会自动暂停并提醒

## 技术实现

### 技术栈
- 纯HTML5 + CSS3 + JavaScript
- 无任何第三方依赖
- 完全前端实现，无需后端服务器

```
debate-timer/

├── styles.css          # 样式文件
```

- **面向对象设计**：使用ES6 Class组织代码
- **状态管理**：集中管理计时器状态

- **模块化数据**：辩论流程数据结构化存储

## 自定义配置

如需修改辩论赛流程或时间，可编辑 `app.js` 文件中的 `debateFlow` 对象：

```javascript
const debateFlow = {
    stages: [
        {
            id: 1,
            name: '第一阶段：开篇立论',
            totalDuration: 6 * 60, // 总时长（秒）
            substages: [
                { 
                    name: '正方一辩立论', 
                    duration: 3 * 60, // 环节时长（秒）
                    desc: '正方一辩阐述本方观点，明确立场' 
                },
                // ...更多子环节
            ]
        },
        // ...更多阶段
    ]
};
```

## 注意事项

1. 本系统不包含投票功能（根据需求设计）
2. 时间到期会自动暂停，需手动点击"下一环节"继续
3. 建议使用大屏幕或投影仪显示，便于全场观看
4. 可通过键盘快捷键（浏览器默认）控制页面
